This endpoint create a Beats.

### HTTP Request

`POST https://sample.herokuapp.com/api/Beats`

### Request Payload

Parameter | Required | Description
--------- | ------- | -----------
beat_name | Required | Name of the beat
vertices | Required | Coordinates of the Beats
